Exercício 1.3

  • Crie uma classe para definir animais de estimação;
  • Crie os atributos comuns para todos os animais deste tipo;
  • Os atributos devem ser privados;
  • Crie 'getters' e 'setters' para todos os atributos que você definir;
  • Crie um 'construtor' que receba os atributos mais importantes;
  • Crie pelo menos 2 objetos instanciando a classe;
  • Defina os atributos dos novos objetos usando os 'setters';
  • Exiba o objeto no console do JavaScript;
  • Exiba pelo menos dois atributos no documento, usando os 'getters'.
  • Crie o diagrama de classe para esta classe.
